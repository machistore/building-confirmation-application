# 建築確認申請書 自動生成システム

## プロジェクト概要
建築基準法に基づく確認申請書（別記第二号様式）をYAMLデータから自動生成するPythonシステム。

## ディレクトリ構成
```
shinsei-generator/
├── CLAUDE.md          # このファイル
├── input/             # 入力YAMLファイル置き場
├── templates/         # Excelテンプレート置き場（将来）
├── src/               # ソースコード
│   ├── generator.py   # メインエントリーポイント
│   ├── calculator.py  # 面積・比率計算
│   └── validator.py   # 入力バリデーション
└── output/            # 生成ファイル出力先
```

## ドメイン知識

### 建蔽率（けんぺいりつ）
- 定義：敷地面積に対する建築面積の割合
- 計算式：建築面積 ÷ 敷地面積 × 100（%）
- 指定建蔽率を超えてはならない

### 容積率（ようせきりつ）
- 定義：敷地面積に対する延べ床面積の割合
- 計算式：延べ床面積 ÷ 敷地面積 × 100（%）
- 指定容積率を超えてはならない

### 延べ床面積
- 各階の床面積の合計

### 別記第二号様式（建築確認申請書）
- 建築基準法施行規則 第1条の3 に基づく書類
- 建築主・設計者・工事施工者・敷地情報・建築物情報を記載

## 技術スタック
- Python 3.x
- 外部ライブラリ：openpyxl（Excel出力）
- 入力フォーマット：YAML（PyYAML使用）

## 禁止事項
- 個人情報（建築主氏名・住所・電話番号）のログ出力禁止
- output/ ディレクトリ以外への生成ファイル出力禁止
- templates/ 内のExcelテンプレートを直接編集しないこと（コピーして使用）

## 開発方針
1. まずテキスト出力（output/result.txt）で計算値を確認する
2. 確認後にExcel出力（openpyxl）へ移行する
3. バリデーションエラーは日本語でわかりやすく表示する
