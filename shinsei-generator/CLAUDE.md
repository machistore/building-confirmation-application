# 建築確認申請書 自動生成システム

## プロジェクト概要
建築確認申請書（別記第二号様式）自動生成システム。
四号建築物（木造2階建て住宅）を対象とする。
入力YAMLを読み込み、テンプレートXLSの各セルに値を書き込み、PDFとして出力する。

## 技術スタック
- Python 3.x
- pyyaml：入力YAMLの読み込み
- xlrd / xlwt / xlutils：XLSテンプレートへの書き込み
- pywin32（win32com）：ExcelからPDFへの変換

## ファイル構成と役割

```
shinsei-generator/
├── CLAUDE.md
├── input/
│   └── sample_project.yaml        # 案件ごとの入力データ
├── templates/
│   └── BPR003_260323.xls          # 申請書テンプレート（編集禁止）
├── src/
│   ├── generator.py               # メイン処理（読込→検証→計算→書込→PDF変換）
│   ├── calculator.py              # 延べ床面積・建蔽率・容積率の計算
│   ├── validator.py               # 必須項目・数値の入力チェック
│   └── pdf_converter.py           # win32com経由のPDF変換
└── output/
    ├── cell_map.yaml              # YAMLキーとExcelセル座標の対応表
    ├── 申請書_出力.xls            # 書き込み済みXLS（生成物）
    └── 申請書_出力.pdf            # PDF（生成物）
```

## 実行方法

```bash
python src/generator.py
```

→ `output/申請書_出力.xls` と `output/申請書_出力.pdf` が生成される。

## ドメイン知識

### 建蔽率（けんぺいりつ）
- 定義：敷地面積に対する建築面積の割合
- 計算式：建築面積 ÷ 敷地面積 × 100（%）
- 指定建蔽率を超えてはならない

### 容積率（ようせきりつ）
- 定義：敷地面積に対する延べ床面積の割合
- 計算式：延べ床面積 ÷ 敷地面積 × 100（%）
- 指定容積率を超えてはならない

### 延べ床面積
- 各階の床面積の合計（calculator.py が計算）

### 別記第二号様式（建築確認申請書）
- 建築基準法施行規則 別記第二号様式に準拠
- 第一面〜第六面 + 注意・別紙 の8シート構成
- 第二面：建築主・代理者・設計者情報
- 第三面：敷地・面積・建蔽率・容積率
- 第四面：建築物別概要（階別床面積）
- 第五面：建築物の階別概要（柱径・天井高・用途別床面積）
- 第六面：建築物独立部分別概要（高さ・階数・構造）

## cell_map.yaml の読み方
- `row` / `col` はすべて **0始まり**（xlrd/xlwt の規則に合わせる）
- `source: data` → `key` をドット区切りでYAML辞書を辿る（例: `"建築主.氏名"`）
- `source: calc` → `generator.py` が渡す計算結果辞書のキーを参照
- `floor_blocks_第五面` セクション → `block_stride` 行ごとに繰り返すブロック構造
- `独立部分_第六面` セクション → `建築物独立部分` リストの先頭1件を書き込む

## 禁止事項
- `templates/` 以下のファイルは**編集禁止**（書式・セル結合が崩れるため）
- 建蔽率・容積率・延べ床面積は `calculator.py` が計算する（手入力禁止）
- 個人情報（建築主氏名・住所・電話番号）のログ出力禁止
- `output/` ディレクトリ以外への生成ファイル出力禁止
